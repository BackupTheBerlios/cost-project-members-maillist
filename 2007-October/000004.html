Hi all

I am now member of the mailing list and have a comment to the question
raised by David Maxwell about converting sefas data to DEF. One of the
suggestion is to include an indication in the SL record of on which
basis the weight estimation is made (whole, gutted...).

I can see David's point if COST is going to include the additional
variance connected to the converting from gutted weight to whole weight.
If this variance is going to be considered, we need not just an
indication but we need the conversion factor and the associated variance
too (assuming that the conversion factor is a mean value of a number of
measures). 

If COST isn't going to consider this variance, I think the information
is irrelevant.

Best regards,

Henrik
