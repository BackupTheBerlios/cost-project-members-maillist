Hi Teunis et al,

We have an issue with the CS SL and CS HL sex fields that we need to discuss. In our sampling, the weight of a box of fish is recorded so there is no information on sex in the SL (SpeciesList) table. When the fish are measured, for some species e.g. plaice or crabs, it is possible to tell which sex they are. This means there are values for sex in the HL (Length record) table. This causes are problem in the current format (11Oct07) because the sex field is part of the key linking the two tables.

Regards, David
