Dear all,

Please find attached some FF data files (CS, CE, CL) which we use for
checking the most basic raising procedures. Be aware of that some values
are not realistic and therefore the results are rather odd (but ok for
checking). There are no "Holes" in the data (=complete match between
CL/CE and CS). The files are in Exchange Format. I hope the files can be
useful for COST.

We are constant developing more sophisticated check data sets capable of
checking corners of stratification and data extrapolation. I will submit
them when they are ready.

Best regards,

Henrik
