> David,

> I use it in unix and never had that problem. Attached is the COSTcore 
> unix version for R 2.8.1. Save it a folder, jump into it and run
>
>    R CMD INSTALL COSTcore_1.3-3.tar.gz
>
> Start R and type
>
>    library(COSTcore)
>
> If you want to use a different library, maybe because you don't have 
> root permission, use
>
>    R CMD INSTALL -l 'path_to_mynewRlib' COSTcore_1.3-3.tar.gz
>
> start R and load with
>
>    library(COSTcore, lib.loc="path_to_mynewRlib")
