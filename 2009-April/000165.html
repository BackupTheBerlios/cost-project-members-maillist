Hi,

The Sweave document is a mixture of code and text. So it takes R code 
that is executed and the result inserted into de document. The text is 
written in LaTeX.

Sweave is a R function, you have it already. What you may need is LaTeX. 
The easiest way to install LaTeX in windows is to do a full installation 
of LyX (www.lyx.org).

The final document shows the text, the code and the code results all 
together.

Attached is a quick and dirty Sweave for the first parts of the script. 
The formatting can be done in several different ways and we have the 
flexibility for format almost everything.

Off course no iterative graphs ...
