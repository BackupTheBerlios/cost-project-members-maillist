Dear COST members,

The draft final report, user manual version 1 and help pages version 1 
have been sent to the European Commission, after a 10 days rally that 
have seen all the elements of the puzzle eventually come into the right 
place.

I must say that it was a useful exercice to be carried out one month 
before the end of the project, because (i) this exercice had to be done 
at some point, and (ii) we can see much more clearly now what is left to 
do in the remaining 4 weeks or so.

Thanks to hard work done by Dorleta and David Hirst, you will see that 
they have been able to test the performance of the methods on a simple 
case over a virtual population. This is a major achievement of the 
project, and it has pointed out some inconsistencies, that will have to 
be sorted out before the end of the project. Next will have to be 
extensive simulation of all methods if we want to validate the tools 
that we will propose to the scientific community.

We'll have to make sure also to have all the packages on a version 1.XX 
within 1 month.

You can find the 3 documents sent to the Commission on the COST website:

   1. http://wwz.ifremer.fr/cost/content/download/45891/347794/file/COST%20Draft%20final%20report.pdf
   2. http://wwz.ifremer.fr/cost/content/download/45892/347797/file/COST%20User%20Manual%20V1.pdf
   3. http://wwz.ifremer.fr/cost/content/download/45893/347800/file/COST%20Help%20Pages%20V1.pdf

Thanks to all for all the hard work. Don't stop your effort now, we 
still need the focus of everyone.

Best regards

Joel
