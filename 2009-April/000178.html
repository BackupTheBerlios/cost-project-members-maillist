Can somebody please find a unix machine with R2.8.1 on, download the 
COSTcore library file and try to install it? It's very easy, all you 
need to do is put the file COSTcore_1.3-3.tar.gz in some directory, 
create a subdirectory called Rlib, then from the directory with the tar 
file in (before starting R) type the command
R CMD INSTALL -l Rlib COSTcore_1.3-3.tar.gz
Then start R and type the command library(COSTcore, lib.loc="Rlib")

Then tell me if you get an error message. I can't load this library on 
any of several machines running both Ubuntu and Suse, but I can load 
every other library I have tried (there are lots at 
http://www.stats.bris.ac.uk/R/). Ernesto can load it, but it seems 
no-one else has tried.

If we can't solve this then we can't finish the project.
