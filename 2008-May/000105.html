Can anyone clarify this:

In Scotland (and probably other countries as well) the size category is 
not nationally defined, instead it is something determined by the 
individual boats. So there could be anything up to 6 size categories on 
any one boat, and size 3 say would not be the same as on any other boat. 
This means that when length samples are taken it is necessary to record 
the number of fish in the size category from which they are taken. This 
usually means one box is taken for length  measurements, and  the number 
of boxes of that size class on the boat is recorded. However, I can't 
see how this will be recorded in the COST format. In the SL record there 
are weight and subsample weight fields, but the comments to the weight 
field say that for market sampling the "catch weight is per definition 
equal to sample weight". What does this mean, and how will the size 
category weight be recorded?

Also, it would be really nice if  there was some more data available. 
Particularly data including discards would be helpful. There isn't any 
at the moment as far as I can tell.
